### 사용하기에 앞서
* 파이썬이 필요합니다
* 설정을 했을 시 나오는 sqlite3 파일을 삭제하면 안됩니다
* 주말에는 사용이 불가능합니다

------------------------
### 처음 시작 가이드
1. 터미널을 엽니다
2. `python3 setup.py` 를 실행합니다
3. 설정이 끝나면 `python3 Timetable.py -e` 를 실행합니다
4. 시간표를 설정합니다
5. 설정이 끝났습니다

### 처음 설정이 끝나면
1. 터미널을 엽니다
2. `python3 Timetable.py -v` 를 실행합니다
3. 온클을 듣습니다

### 인수 목록
* `-v`: 시간표를 확인합니다
* `-d`: 시간표를 삭제합니다
* `-e`: 시간표를 설정합니다
